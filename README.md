## ✨ Features
- **Dynamic Game Modes:** Experience 'Speed', 'Pulse', 'Trick', and '50/50 Chaos' rounds as your streak gets higher.
- **Cinematic Experience:** Built with custom animations using `motion/react` and a sleek monochromatic UI.
- **Micro-interactions:** Audio cues and animated state transitions create an exceptional user feel.
- **Social Sharing:** Easily challenge friends and share your highest streaks natively.
- **SEO & Vercel Analytics:** Fully optimized meta tags, Open Graph previews, and built-in tracking for live visitors.

## 🛠 Tech Stack
- **Framework:** React 19 + TypeScript + Vite
- **Styling:** Tailwind CSS v4
- **Animations:** Motion
- **Analytics:** `@vercel/analytics`
- **Icons:** Lucide React

## 🚀 Getting Started

Follow these instructions to run the project locally.

### Prerequisites
- [Node.js](https://nodejs.org/) (v18 or higher)
- NPM or Yarn installed

### Installation

1. **Clone the repository:**
   ```bash
   git clone https://github.com/Sumitagarwal-i/real-or-fake-game.git
   cd real-or-fake-game
   ```

2. **Install dependencies:**
   ```bash
   npm install
   ```

3. **Running the application:**
   ```bash
   npm run dev
   ```

4. **Build for production:**
   ```bash
   npm run build
   ```

## 🎮 How to Play
- Click **Start Game** and read the statement on the screen rapidly.
- Decide if the statement is **REAL** or **FAKE** before the timer runs out!
- As your score increases, you'll encounter unpredictable "Speed Rounds" and "Chaos Rounds".
- Your highest score and streak are automatically saved locally.
